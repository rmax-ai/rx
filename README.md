# rx

## Build Instructions

1. Install the Rust toolchain (if not already installed):
   ```bash
   curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
   # or use your preferred Rust installer
   ```
2. Build the project with Cargo:
   ```bash
   cargo build --release
   ```

Run `cargo run` to execute the binary or `cargo test` to run the test suite.
